# Vue.js SSR Template 

> A Dead Simple Vue.js Server Side Render Template 

## Demo

[Vue.js SSR Template Demo](http://ssr.ccforward.net/)

## Build Setup

``` bash
# npm
npm install

# yarn
yarn

# develop
npm run dev

# deploy
npm run deploy

# build 
npm run build
```
